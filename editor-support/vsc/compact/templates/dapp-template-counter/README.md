# Counter

This is the simplest Midnight smart contract example

## Quick start

Execute all actions from [Running tests](#running-tests)

## Building and running tests

To setup environment you need to:
1. install yarn
2. configure credentials

### using yarn

In terminal execute commands:

```sh
yarn install  # Install dependencies  
yarn compact  # Transpile the Compactc smart contract
yarn build    # Build TS source files and copy smart contract
yarn test     # Run tests
```

## Formatting source code

Run Linter:

```sh
yarn lint
```
